
This game will work properly only at 100% screen size (DPI) in system Control Panel.
If you have HI-res display, you should open properties of executable file and turn on compatibility feature that disables scaling.

Эта игра работает правильно только при размере экрана 100% (DPI) в системной Панели Управления.
Если у вас дисплей высокого разрешения, то вам следует открыть свойства исполняемого файла и включить режим совместимости – галочку, отключающую масштабирование.
